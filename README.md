# Private-007

A love-themed (black + pink) mini website for viewing screenshots and audio clips.

## Features
- Three openable sectors:
  - `legend himself` (top-most)
  - `proof's`
  - `audio proofs`
- Custom heading text:
  - `anay diddy files`
  - `Operation anay`
  - `witness the diddy of oxf <3`
- Upload controls are removed/locked for visitors.

## Run locally
Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
